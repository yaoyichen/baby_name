# 五格三才数理分析系统文档

## 一、基本概念

五格三才是中国传统姓名学的核心体系，通过统计姓名各部分的笔画数，计算出五个格位，再结合五行（木火土金水）的生克关系，评估姓名的吉凶运势。

---

## 二、五格计算架构

### 2.1 双字名（三字姓名：姓 + 名一 + 名二）

设姓氏笔画数为 **S**，名一笔画数为 **A**，名二笔画数为 **B**。

| 格位 | 公式 | 运势周期 | 核心含义 |
|------|------|----------|----------|
| 天格 | S + 1 | 祖宗运 | 先天遗传、长辈福荫（受姓氏所限，自身无法更改） |
| 人格 | S + A | 主运 | 核心运势、性格特质、中年成就（权重最高） |
| 地格 | A + B | 前运 | 36 岁之前的青年运势、基础建设 |
| 总格 | S + A + B | 后运 | 中晚年总运、最终成就与人生定局 |
| 外格 | B + 1 | 副运 | 社交人缘、外部环境支持 |

### 2.2 单字名（两字姓名：姓 + 名一）

单字名引入**假添一数**规则，补齐缺失的名二（默认视为 1 画）。

设姓氏笔画数为 **S**，名一笔画数为 **A**。

| 格位 | 公式 | 特殊架构说明 |
|------|------|-------------|
| 天格 | S + 1 | 固定不变 |
| 人格 | S + A | 核心主运 |
| 地格 | A + 1 | 触发假添一数 |
| 总格 | S + A | 等于人格（一荣俱荣，一损俱损） |
| 外格 | 1 + 1 = 2 | 固定为 2（木），不具备参考价值，退出加权计算 |

## 四、基础分加权计算（双体系）

为解决天格"姓氏原罪"及单字名"权重重叠"问题，采用两套独立的加权体系。

### 4.1 双字名加权系数

弱化天格（先天定数）影响，强化人格与地格（后天努力）。

$$Score_{base} = T_{score} \times 0.05 + R_{score} \times 0.45 + D_{score} \times 0.25 + Z_{score} \times 0.20 + W_{score} \times 0.05$$

| 格位 | 权重 |
|------|------|
| 人格 | 45%  |
| 地格 | 25%  |
| 总格 | 20%  |
| 天格 | 5%   |
| 外格 | 5%   |

### 4.2 单字名加权系数

因单字名总格与人格重复，且外格固定为 2（无效格），重新分配核心权重。

$$Score_{base} = T_{score} \times 0.05 + R_{score} \times 0.60 + D_{score} \times 0.35$$

| 格位 | 权重 |
|------|------|
| 人格 | 60%  |
| 地格 | 35%  |
| 天格 | 5%   |
| 外格 | 0%（退出计算） |
| 总格 | 0%（与人格重复，不另计） |

## 五、三才五行动力学修正

### 5.1 尾数 → 五行映射

| 尾数 | 五行 |
|------|------|
| 1、2 | 木   |
| 3、4 | 火   |
| 5、6 | 土   |
| 7、8 | 金   |
| 9、0 | 水   |

五行关系：
- **相生**：木→火→土→金→水→木
- **相克**：木克土，土克水，水克火，火克金，金克木

三才判定（天格尾→人格尾→地格尾）：
- **吉**（触发加分）：我生、生我、同行（相同五行）
- **凶**（触发减分）：我克、克我

### 5.2 动态边界比例修正

摒弃绝对值加减，采用**剩余空间比例修正法**，确保分数永远在合理区间 [20, 100] 内波动。

**情形一：三才相生**
（天人、人地均不含相克，且至少有一组相生/同行）

$$Score_{final} = Score_{base} + (100 - Score_{base}) \times 0.3$$

**情形二：三才相克**
（天人或人地存在任一组相克）

$$Score_{final} = Score_{base} - (Score_{base} - 20) \times 0.2$$

**情形三：三才平和**
（无相生相克，极罕见）

$$Score_{final} = Score_{base}$$

## 六、连凶惩罚与最终定级

### 6.1 核心格连凶惩罚（Structural Penalty）

若名字的**三大核心格（人格、地格、总格）**中，出现 **2 个或以上**的「大凶」（即单格基础分 ≤ 20），说明名字结构性崩坏，触发系统惩罚：

$$Score_{final} = Score_{final} \times 0.9$$

> 单字名中，总格与人格数值相同，因此判断时以 [人格, 地格, 人格] 三项参与统计。

### 6.2 最终评分等级

| 分数区间 | 等级 | 颜色标识 | UI 提示文案 |
|----------|------|----------|------------|
| 90 ~ 100 | 上吉 | 翠绿色 `#2E7D32` | 佳名天成，底蕴深厚，三才五格搭配绝佳。 |
| 80 ~ 89  | 中吉 | 湖蓝色 `#0277BD` | 运势亨通，结构稳健，为不可多得的好名。 |
| 65 ~ 79  | 平顺 | 琥珀色 `#F57C00` | 吉凶相伴，需凭后天定力，整体无大碍。 |
| 50 ~ 64  | 偏凶 | 砖红色 `#C62828` | 运势偶有波折，建议结合生辰八字综合考量。 |
| 0 ~ 49   | 大凶 | 深暗红 `#b71c1c` | 基础不稳，阻力较大，建议重新斟酌选字。 |

## 七、计算示例

### 示例：姚（姓）+ 清（名一）+ 朝（名二）

- 姚：9画，清：11画，朝：12画

| 格位 | 计算 | 数值 | 尾数五行 | 等级 |
|------|------|------|----------|------|
| 天格 | 9+1  | 10   | 水（0=水）| 凶（40） |
| 人格 | 9+11 | 20   | 土（0=土？→水）| 大凶（20） |
| 地格 | 11+12| 23   | 火（3=火）| 大吉（100） |
| 总格 | 9+11+12| 32 | 木（2=木）| 大吉（100） |
| 外格 | 12+1 | 13   | 火（3=火）| 大吉（100） |

**三才**（天格尾→人格尾→地格尾）：水(10)→水(20)→火(23)，人地相克（水克火），触发减分。

**基础分（双字名）**：
$$40\times0.05 + 20\times0.45 + 100\times0.25 + 100\times0.20 + 100\times0.05 = 2 + 9 + 25 + 20 + 5 = 61$$

**三才修正**（人地相克，hasKe=true）：
$$61 - (61-20)\times0.2 = 61 - 8.2 = 52.8 ≈ 53$$

**连凶惩罚检查**（人格=20≤20，地格=100，总格=100，仅1个大凶）：不触发

**最终分数**：**53**（偏凶）
